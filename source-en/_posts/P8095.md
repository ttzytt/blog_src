# 1: Problem Statement
There are $N$ cows and $M$ cereal types, one box of each type. Every cow has a first and second choice. A cow takes its first choice if available; otherwise it takes its second choice. Find:
1. The minimum number of cows that receive no cereal.
2. An ordering of cows that achieves this minimum.

# 2: Analysis
## 2.1 First Question
The first question is a standard maximum bipartite matching problem, solvable with the Hungarian algorithm. This solution focuses on the ordering in the second question.

## 2.2 Second Question
It is not enough to output cows matched to their first choice, then cows matched to their second choice, and finally unmatched cows. Consider:
```
1 (cow) -> [1 (first), 2 (second)]
2 (cow) -> [1 (first), 3 (second)]
3 (cow) -> [3 (first), 4 (second)]
```
Ordering `1 2 3` lets every cow obtain cereal: cow 1 takes 1, cow 2 takes 3, and cow 3 takes 4. Ordering `1 3 2` makes cow 2 lose both 1 and 3.

The second-choice cows must therefore be output in an order that preserves the maximum matching. This is an influence chain similar to topological sorting. A cow matched to its first choice may occupy another cow's first choice, forcing that cow to use its second choice; that second choice may in turn occupy another cow's first choice.

# 3: Algorithm
The beginning of each chain is a cow matched to its first choice whose first choice is also another cow's first choice. Put all such cows into the output first. Maintain `inve[cereal]`, the cows whose first choice is that cereal. `invmatched[cow]` records the cereal finally matched to the cow.

For every first-choice cow, enqueue affected cows that are matched to their second choice. Then process this queue FIFO, repeatedly following the second-choice cereal to cows whose first choice it is. Finally output unmatched cows.

## Code
```cpp
#include<bits/stdc++.h>
using namespace std;
const int MAXN = 2e5 + 10;
int n, m;
vector<int> e[MAXN], inve[MAXN];
queue<int> q;
int vised[MAXN], matched[MAXN], invmatched[MAXN];

bool found(int cur){// Hungarian algorithm.
    for(int nex:e[cur]){
        if(vised[nex]) continue;
        vised[nex] = true;
        if(!matched[nex] || found(matched[nex])){
            matched[nex] = cur;
            invmatched[cur] = nex;
            vised[nex] = false;
            return true;
        }
    }
    return false;
}

int main(){
    int match_cnt = 0;
    scanf("%d%d",&n,&m);
    for(int i = 1; i<=n; i++){
        int f,s;
        scanf("%d%d",&f,&s);
        e[i].push_back(f);   // e[i][0] is cow i's first choice.
        e[i].push_back(s);   // e[i][1] is cow i's second choice.
        inve[f].push_back(i);// Cows whose first choice is f.
    }
    for(int i = 1;i <= n; i++) if(found(i)) match_cnt++;
    printf("%d\n", n - match_cnt);// Hungry cows = all cows - matched cows.

    for(int i = 1; i<=n; i++){
        if(invmatched[i] != e[i][0]) continue;
        printf("%d\n",i);
        for(int cur:inve[e[i][0]]){
            if(invmatched[cur] == e[cur][1]) q.push(cur);
        }
    }
    while(!q.empty()){
        int cur = q.front();
        printf("%d\n",cur);// FIFO outputs cows earlier in the influence chain first.
        q.pop();
        for(int nex:inve[e[cur][1]]){
            if(invmatched[nex] == e[nex][1]) q.push(nex);
        }
    }
    for(int i = 1; i<=n; i++)
        if(!invmatched[i]) printf("%d\n",i);// Finally output unmatched cows.
    system("pause");
}
```
