# C. Train and Queries

For a station sequence $u$, trains move only from left to right. For each query $(a,b)$, determine whether a train can start at station $a$ and reach station $b$. Store the first position `first[x]` and last position `last[x]` of every station number. The answer is YES exactly when both stations exist and `first[a] <= last[b]`.

```cpp
map<int,int> first,last;
for(int i=1;i<=n;i++){
  cin>>a[i];
  if(!first[a[i]]) first[a[i]]=i;
  last[a[i]]=i;
```

# D. Not a Cheap String

The price of a lowercase string is the sum of each letter's alphabet position. To reduce the price to at most $p$ while deleting as few characters as possible, delete the most expensive available letters first. Count each character in a bucket, traverse buckets from `z` down to `a`, and decrement counts until the price is small enough. Then scan the original string and emit characters whose remaining bucket count is positive.

```cpp
map<char,int> bucket;
long long price=0;
for(char c:s){ bucket[c]++; price+=c-'a'+1; }
for(auto it=bucket.rbegin(); price>p; ++it){
  while(it->second && price>p){ it->second--; price-=it->first-'a'+1; }
}
for(char c:s) if(bucket[c]--) cout<<c;
cout<<'\n';
```

# E. Split Into Two Sets

Given $n$ pairs, determine whether they can be divided into two sets so that no number appears twice in one set. A greedy assignment fails. Regard every number as a graph vertex and every pair as an edge. At each vertex, incident edges must alternate between the two sets. Therefore a vertex of degree greater than 2 is impossible; every path is valid; and a cycle is valid exactly when it has even length.

Build the graph without duplicate edges, reject self-loops and degrees above two, and DFS-color each component alternately. A conflict means an odd cycle and produces `NO`; otherwise output `YES`.

```cpp
bool dfs(int v,int parent,int color){
  for(auto &e:graph[v]){
    if(e.to==parent) continue;
    if(e.color==-1) e.color=!color;
    else if(e.color==color) return false;
    if(!dfs(e.to,v,!color)) return false;
  }
  return true;
}
```

# F. Equate Multisets

For each element of multiset $b$, operations are multiplication by 2 or floor division by 2. In binary these are left and right shifts. Trailing zeroes are irrelevant, so remove them from both multisets. A normalized value $a$ can be obtained from normalized $b$ exactly when the binary representation of $a$ is a prefix of that of $b$.

Sort normalized `a` and put normalized `b` in a max-heap. Compare their largest elements:

1. If `a_max > b_max`, answer NO.
2. If they are equal, remove both.
3. If `a_max < b_max`, divide `b_max` by 2 and push it back.

```cpp
while(!b.empty()){
  int lb=b.top(); b.pop();
  int la=a.back();

  if(la<lb){ b.push(lb/2); }
  else a.pop_back();
}
cout<<"YES\n";
```
