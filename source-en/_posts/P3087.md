# 1: Understanding the Problem

There are $cow$ cows and $type$ adjective classes. Class $i$ has $num_i$ adjectives, and the $j$th adjective in class $i$ is $adj_{i,j}$. Every cow is described by one adjective from each class in order. We are told to remove $n$ of the $cow$ cows. Among the remaining $cow-n$ cows sorted lexicographically, which cow is in position $k$?

# 2: Analysis and Reformulation
## 2.1: Connection with Number Systems

In the sample, $n=3$ and $k=7$, with the following adjectives:

| $adj_{i,j}$ | $j=1$ | $j=2$ | $j=3$ |
|:--|:--|:--|:--|
| $i=1$ (first class) | large | small | N/A |
| $i=2$ (second class) | brown | white | spotted |
| $i=3$ (third class) | noisy | silent | N/A |

Lexicographic comparison gives earlier positions greater influence: compare the first character first, then the second, and finally the third. Thus the first adjective class has the greatest influence, followed by the second, then the third.

This is closely related to a positional number system. In $(123456789)_{10}$, digit 1 represents $100000000$, digit 2 represents $20000000$, and a digit $x$ at position $i$ represents $10^{i-1}x$. In base $k$, it represents $k^{i-1}x$.

## 2.2: Solving the Simplified Problem

First assume every adjective class contains exactly $k$ adjectives. Sort every class lexicographically and store the ranks in `rank[i][j]`. This maps each adjective to a digit in a base-$k$ number system. The smaller the adjective class index, the greater its lexicographic influence, so it corresponds to a higher digit position.

After mapping adjectives to digits, convert a decimal rank to base $k$, then map the digits back to adjectives.

For example, suppose the three classes are `{a,b}`, `{c,d}`, and `{e,f}`. With ranks starting at 0, the rank table is:

| $rank_{i,j}$ | $j=0$ | $j=1$ |
|:--|:--|:--|
| $i=1$ | a | b |
| $i=2$ | c | d |
| $i=3$ | e | f |

To find the cow at lexicographic position 3, convert 3 to binary, $(011)_2$, reverse it to $(110)_{2\text{ reversed}}$, and map the digits back to classes. The answer is `a, d, f`.

## 2.3: The Original Problem

In the original problem, the number of adjectives differs by class, so each digit position has a different base. In the sample, the bases from the least significant class upward are $2,3,2$.

To convert a decimal rank, precompute the decimal value represented by each position. The first position has weight 1; each preceding position's weight is multiplied by the number of values in that position. Then repeatedly select the digit for each position:

```cpp
// k is the decimal number.
// weight_in_pos[i] is the decimal value of position i, counted from the highest position.
// i is the current position, also counted from the highest position.
for (int i = 1; i <= adj_num; i++) {
    cout << adj_by_pos[i][k / weight_in_pos[i]] << " ";
    k %= weight_in_pos[i];
}
```

Finally, the requested rank is among the remaining cows rather than all cows. Compute the ranks of the $n$ removed cows. For every removed cow whose rank is less than or equal to $k$, increment $k$; then convert the resulting rank back to adjectives.

# Code and Details

```cpp
#include <bits/stdc++.h>
using namespace std;
#define ll long long
int n, k;
int adj_num = 0;
vector<string> str[105];          // Adjectives describing cow i.
vector<string> adj_by_pos[35];    // All adjectives appearing at position i.
set<string> is_appear[35];        // Determine whether an adjective appears at position i.
int weight_in_pos[35];            // Decimal weight represented by each position.
map<string, int> rank_in_pos[35]; // Lexicographic rank of string j at position i.
int cow_rank[105];                // Ranks of cows removed by Farmer John.
bool debug = false;               // Debug switch.
void mapping()                    // Sort each adjective class and store ranks.
{
    for (int i = 1; i <= adj_num; i++) {
        int rank = 0;
        for (auto j : adj_by_pos[i]) {
            rank_in_pos[i][j] = rank;
            if (debug) cout << j << " rank = " << rank << " i = " << i << endl;
            rank++;
        }
    }
}

int get_pos(int cow_id) {
    int ans = 0;
    for (int i = 1; i <= adj_num; i++)
        ans += weight_in_pos[i] * rank_in_pos[i][str[cow_id][i - 1]];
    return ans + 1; // The value may be 0, but ranks start at 1.
}

int main() {
    ios::sync_with_stdio(false);
    cin >> n >> k;
    string temp_str;
    for (int i = 1; i <= n; i++) {
        cin >> temp_str;
        while (temp_str != "no") cin >> temp_str;
        int adj_pos = 1;
        while (1) {
            cin >> temp_str;
            if (temp_str == "cow.") break;
            str[i].push_back(temp_str);
            if (!is_appear[adj_pos].count(temp_str)) {
                adj_by_pos[adj_pos].push_back(temp_str);
                is_appear[adj_pos].insert(temp_str);
            }
            if (i == 1) adj_num++;
            adj_pos++;
        }
    }
    for (int i = 1; i <= adj_num; i++) sort(adj_by_pos[i].begin(), adj_by_pos[i].end());

    weight_in_pos[adj_num + 1] = 1;
    adj_by_pos[adj_num + 1].push_back("temp");
    for (int i = adj_num; i >= 1; i--)
        weight_in_pos[i] = weight_in_pos[i + 1] * adj_by_pos[i + 1].size();
    mapping();
    for (int i = 1; i <= n; i++) cow_rank[i] = get_pos(i);
    sort(cow_rank + 1, cow_rank + n + 1);
    for (int i = 1; i <= n; i++) {
        if (cow_rank[i] <= k) k++;
        else break;
    }
    k--;
    for (int i = 1; i <= adj_num; i++) {
        cout << adj_by_pos[i][k / weight_in_pos[i]] << " ";
        k %= weight_in_pos[i];
    }
    system("pause");
}
```
